Medical Demand Forecasting System
Live App:
https://futureml01-nm5wi243oemqfoat4p6y5w.streamlit.app

1) Project Overview:

This project is a GUI-based Medical Demand Forecasting System developed using Python and Tkinter. It allows pharmacies or healthcare businesses to:

 Forecast medicine demand
 Compare multiple medicines
 Export results to CSV and Excel
 Visualize forecasts with graphs
 Automatically generate observations


2) The system supports both:

 Real CSV pharmacy datasets
 Auto-generated sample data (if no dataset is loaded)



3) Machine Learning Models Used:

The system supports three forecasting models:

1. XGBoost (Recommended)
2. Linear Regression
3. ARIMA



4) Data Export:

 Export forecast results to CSV
 Export forecast results to Excel



5) Technologies Used:
  Python
  Tkinter (GUI)
  Pandas
  Matplotlib
  XGBoost
  Statsmodels (ARIMA)
  Scikit-learn (Linear Regression)




6) How It Works:

1. Load pharmacy CSV dataset (optional).
2. Select medicine.

3. Choose forecasting model (XGBoost recommended).
4. Select forecast days (7 or 30 days).
5. Run forecast.
6. View units, revenue, comparison, and observations.



If no dataset is loaded, the system automatically generates realistic sample pharmacy data.


---

7) Conclusion:

This Medical Demand Forecasting System combines machine learning with practical business analysis. XGBoost is recommended as the primary model due to its superior forecasting accuracy and industry relevance.
